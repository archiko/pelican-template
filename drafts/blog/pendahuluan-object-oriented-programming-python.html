Pendahuluan object oriented programming python

 python oop

Object oriented programming (OOP) Adalah sebuah paradigma atau cara berpikir pemogramman yang menggunakan konsep object yang mana didalam object tersebut terdapat class, attributes dan methods.

Semisal disebuah game terdapat banyak karakter dan setiap karakter tersebut dibungkus dalam sebuah template (class),kenapa membutuhkan template? gini kita asumsikan setiap karakter itu mempunyai role (attribute) masing-masing dan mempunyai aksi seperti menyerang dan bertahan (method). tentu jika kita tidak menggunakan template maka pembuatan setiap karakter itu akan membutuhkan waktu yang lama, karena semisal kita punya puluhan karakter. maka dari itu adanya konsep oop ini membuat kode kita menjadi sederhana dan dapat digunakan kembali (reuseable).

contoh kita mempunyai sebuah class yang bernama character didalamnya terdapat role dan name sebagai attribute. ia juga mempunyai dua aksi yaitu bertahan dan menyerang sebagai method.

class character:

    def __init__(self,nama,role):
        self.name = nama
        self.role = role

    def attack(self):
        return 'menyerang'

    def defense(self):
        return 'bertahan'

xhiko = character('xkira','assasin')
xkira = character('xhiko','support')

nah variabel xhiko dan xkira itu adalah sebuah object, sedangkan class character itu sebagai template, didalamnya juga terdapat attribute dan opsi aski mereka masing-masing.

kita bisa menampilkan seluruh attribute dan methodnya seperti dibawah ini.

xhiko = character('xhiko','assasin')
xkira = character('xkira','support')

print('nama: '+xhiko.name,' role: '+xhiko.role)
print(xhiko.attack())

print('nama: '+xkira.name,' role: '+xkira.role)
print(xkira.defense())

maka outputnya akan seperti ini


nama: xhiko  role: assasin
menyerang
nama: xkira  role: support
bertahan



Kesimpulan

jadi intinya oop itu dapat mempermudah penyusunan dan pembuatan sebuah Program, khususnya yang berskala besar.
